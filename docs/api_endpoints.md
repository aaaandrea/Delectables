#API Endpoints

HTML API
Root
- GET / - loads React web app

JSON API
Users
- POST /api/users
- GET /api/user/:id

Session
- POST /api/session
- DELETE /api/session

Recipes
- GET /api/recipes
- GET /api/recipes/:id
- POST /api/recipes
- PATCH /api/recipes/:id
- DESTROY /api/recipes/:id
- POST /api/recipes/comments
